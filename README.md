    - 컴포넌트 관련

        - button 이용시 elements 폴더 Button 숙지 후 재이용 할 것

            - url이 home일 경우 전체화면과 현재 저장된 repository 확인가능하게
            - 이외에는 navbar 역할
        - Main.jsx는 빈 컴포넌트

    - 리덕스 관련
        - isLoading 관련 통일할 것

    - 공통 관련
        - console.log() 대신 testLogger 이용할 것
        - style 관련 styles -> theme 참고하여 스타일 설정
